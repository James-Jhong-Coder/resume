履歷簡述
簡述：簡介專長與技能。
經歷：做過什麼、有什麼成效或貢獻，有具體數字會更好。通常最值得好好探討的是前面提到的成果和成就。
學歷：若經歷豐富，可以只寫最高學歷。通常有第一份工作經驗後，學歷就不太重要了。
技能：擅長的技能或技術。
作品：展現實力、喜愛的技術、專業度。
自傳：表達人格特質。

檢視職涯歷程的優點，選擇 3 - 5 個重點來寫。
利用這 3 - 5 個重點，為自己撰寫 3 句話作為履歷的簡述。
